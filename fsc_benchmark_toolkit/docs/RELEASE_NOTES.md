# Release Notes

## v0.1.0

Initial executable scaffolding for the FSC Benchmark repository.

Included:

- JSONL input schema.
- FSC and FSC-hard metric computation.
- sample-level classification.
- grouped metrics.
- Markdown report generation.
- synthetic fixture records.
- documentation for taxonomy, metrics, validation, and task ecosystems.

Not included:

- final benchmark tasks.
- empirical model outputs.
- official leaderboards.
- prevalence estimates.

These materials will be released upon paper acceptance or in a future benchmark release.
