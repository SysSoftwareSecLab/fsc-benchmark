# Manifest

- `README.md`: repository overview and quick-start guide.
- `pyproject.toml`: Python packaging configuration.
- `src/fsc_benchmark/`: core Python package.
- `examples/generations/demo_generations.jsonl`: synthetic generation records.
- `examples/tasks/demo_tasks.json`: synthetic task metadata.
- `docs/TAXONOMY.md`: FSC taxonomy and boundaries.
- `docs/METRICS.md`: metric definitions.
- `docs/INPUT_SCHEMA.md`: input schema.
- `docs/VALIDATION_GUIDE.md`: validation recommendations.
- `docs/TASK_ECOSYSTEMS.md`: task ecosystem guidance.
- `tests/`: unit tests.
- `reports/`: generated report examples.

The repository intentionally excludes final benchmark data and empirical outputs.
